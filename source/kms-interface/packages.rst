KMS Thrift Interfaces Documentation
===================================

The :java:package:`com.kurento.kms.thrift.api` package documents
the java side of the Thrift interfaces between the Kurento Media
Server and java code. This interface manages low level communications
with a Kurento Media Server.

The Java documentation of the Thrift interface is cumbersome to read,
and this API is only recommended for extending or controlling the Kurento
Media Server. The :doc:`KMF Content API </kmf-content-api/packages>` is the
recommended way to program the Kurento Framework for server side Java
applications, and the :doc:`KMF Media API </kmf-media-api/packages>` for
low level handling.
